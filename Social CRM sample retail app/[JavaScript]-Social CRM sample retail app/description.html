Introduction

The Social CRM sample app is a sample implementation of the app described in the Windows design inspiration Retail apps: Social CRM idea book. You can use the Social CRM sample app to see how the design concepts of a retail social customer relationship management app are implemented. The Social CRM sample app includes UI based on the idea book including examples of Toast notifications, Share, and Live tiles.

Build the sample

1.      Extract the zipped files into a local directory.

2.      Start Visual Studio and select File > Open > Project/Solution.

3.      Go to the directory in which you unzipped the sample. Double-click the Visual Studio solution file named Contoso.sln.

4.      Select Build > Rebuild Solution to build the sample.

This solution can be created or compiled using the Pro, Enterprise, Ultimate, or Express versions of Visual Studio.
