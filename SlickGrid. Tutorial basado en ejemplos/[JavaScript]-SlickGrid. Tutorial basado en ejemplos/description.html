SlickGrid es uno de los grid cliente de software libre más ampliamente utilizados (sino el que más después de que jqGrid haya pasado a ser de pago).

 

El código son una serie de ejemplos de uso del SlickGrid a modo de turtorial.

 

EjemploBasico.html

 En este ejemplo muestro como crear un grid, cargar y modificar datos, y un primer acercamiento al manejo de eventos y la ordenación.

Se puede encontrar una descripción detallada del ejemplo en los artículos:

Ejemplo básico (I) - Creando el primer grid

Ejemplo básico (y II) - Añadiendo funcionalidad

 

FormatoColumnas.html

Muestra como aplicar estilos a las cabeceras y columnas y cómo crear funciones de formateo para tener un control total sobre la forma de visualizar los datos.

Se puede encontrar una descripción detallada del ejemplo en los artículos:

Dando formato a las celdas (I) - Formato básico

Dando formato a las celdas (y II) - Formato avanzado

 

EdicionBasica.html

Cómo habilitar la edición y crear editores de celda personalizados.

Se puede encontrar una descripción detallada del ejemplo en los artículos:

Habilitando la edición (I) - La teoría

Habilitando la edición (y II) - Creando los editores

 

CargaCeldaAsincrona.html

Muestra cómo cargar datos de forma asíncrona usando la propiedad asyncPostRender y cómo crear una caché para acelerar la carga asíncrona de datos.

Se puede encontrar una descripción detallada del ejemplo en los artículos:

Carga asíncrona de celdas (I) - La propiedad asyncPostRender

Carga asíncrona de celdas (y II) - Generando una caché de datos

 

DataProvider.html

Cómo crear un proveedor de datos que haga de interfaz entre los datos y el SlickGrid. El proveedor de datos nos permitirá no sólo controlar los datos a mostrar en el grid, si no también la forma de mostrarlos.

Se puede encontrar una descripción detallada del ejemplo en los artículos:

Proveedor de Datos (I) - Creando un proveedor de datos

Proveedor de Datos (y II) - El método getItemMetadata

 

 

DataView.html

Cómo utilizar el proveedor de datos DataView y sus características, como las de ordenación, filtrado o paginación.

Se puede encontrar una descripción detallada del ejemplo en los artículos:

DataView (I) - Introducción y uso

DataView (II) - Ordenación simple y múltiple

DataView (III) - Filtros de datos

DataView (y IV) - Paginación
