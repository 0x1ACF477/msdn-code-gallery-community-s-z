## Description

<h1>Introduction</h1>
<p><span style="font-size:small"><em>Now a days all the websites are developed as responsive. So hereafter we can also develop our sharepoint application as responsive.&nbsp;</em></span></p>
<h1>Description</h1>
<p><span style="font-size:small">Sharepoint sites are mostly used in bussiness sides for workflow projects and teamsites to maintain documents. This site will give a good look in system only. Not in tablets and mobile devices.</span></p>
<p><span style="font-size:small">Even though we designed a responsive content for our site, the default master page will not allow the site to be responsive.</span></p>
<p><span style="font-size:small">But I customized this sharepoint master page as responsive using bootstrap javascript and styles.</span></p>
<p><span style="font-size:small">By using this custom sharepoint master page template, we can make our site responsive to view better in all the devices, not only in system.&nbsp;</span></p>
<p>&nbsp;</p>
<p><strong style="font-size:2em"><em><span style="text-decoration:underline">Creating Master Page</span></em></strong></p>
<p><span style="font-size:medium"><strong>Step:1</strong></span></p>
<p><span style="font-size:small">Download the&nbsp;<strong><em>Responsive Master Page Template.zip</em></strong>&nbsp;here.</span></p>
<p><span style="font-size:medium"><strong>Step:2</strong></span></p>
<p><span style="font-size:small">Go to your&nbsp;<strong><em>site collection</em></strong>&nbsp;and navigate to&nbsp;<strong><em>Site Settings</em></strong></span></p>
<p><span style="font-size:medium"><strong>Step: 3</strong></span></p>
<p><span style="font-size:small">(Make sure you have enabled the&nbsp;<strong><em>publishing future</em></strong>&nbsp;in site collection level)</span></p>
<p><span style="font-size:small">Select the&nbsp;<strong><em>Master pages and page layouts</em></strong>&nbsp;option under Galleries</span></p>
<p>&nbsp;</p>
<p>&nbsp;</p>
<p><img src="http://www.codeproject.com/KB/Articles/1037937/1.png" alt="" width="657" height="237"></p>
<p>&nbsp;</p>
<p>&nbsp;</p>
<p>&nbsp;</p>
<p>&nbsp;</p>
<p><span style="font-size:medium"><strong>Step: 4</strong></span></p>
<p><span style="font-size:small">Open the downloaded zip file and copy</span></p>
<p><span style="font-size:small"><strong><em>Responsive Master Page Template-&gt;MasterPage-&gt;bootstrap</em></strong>&nbsp;and upload it under the master page list</span></p>
<p><span style="font-size:small">(Make sure you have uploaded the &ldquo;bootstrap&rdquo; folder. Inside the folder, one master file will be there)</span></p>
<p>&nbsp;</p>
<p><img src="http://www.codeproject.com/KB/Articles/1037937/2.png" alt="" width="472" height="203"></p>
<p>&nbsp;</p>
<p><span style="font-size:medium">Step: 5</span></p>
<p><span style="font-size:small">Click<strong>&nbsp;<em>All Site Content</em></strong>&nbsp;link from Quick Launch pane.&nbsp; (Site Collection Level)</span></p>
<p><img src="http://www.codeproject.com/KB/Articles/1037937/3.png" alt="" width="658" height="196"></p>
<p>&nbsp;</p>
<p><span style="font-size:medium"><strong>Step: 6</strong></span></p>
<p><span style="font-size:small">Select&nbsp;<strong><em>Style Library</em></strong>&nbsp;list to upload the bootstrap styles</span></p>
<p><img src="http://www.codeproject.com/KB/Articles/1037937/4.png" alt="" width="565" height="287"></p>
<p><span style="font-size:medium"><strong>Step: 7</strong></span></p>
<p><span style="font-size:small">Open the downloaded zip file and copy</span></p>
<p><span style="font-size:small"><strong><em>Responsive Master Page Template-&gt;StyleLibrary-&gt;bootstrap</em></strong>&nbsp;and upload it under the Style Library list</span></p>
<p><img src="http://www.codeproject.com/KB/Articles/1037937/5.png" alt="" width="546" height="277"></p>
<p><span style="font-size:small">(Make sure you have uploaded the &ldquo;bootstrap&rdquo; folder. Inside the folder, img, styles, scripts folders will be there.)</span></p>
<p>&nbsp;</p>
<p><span style="font-size:medium"><strong>Step: 8</strong></span></p>
<p><span style="font-size:small"><strong><em>Check In</em></strong>&nbsp;all the uploaded files and folders and&nbsp;<strong><em>approve</em></strong>&nbsp;it.</span></p>
<p><span style="font-size:small">Now your responsive Master page is ready.</span></p>
<p><span style="font-size:small">Note: Set this master page as default master page to apply this responsive template to all sub site and site pages.</span></p>
<p><span style="font-size:small">Please follow the&nbsp;<strong><em>below steps</em></strong>&nbsp;to apply this master page for&nbsp;<strong><em>individual site pages</em></strong>. (Better apply this master page for user level pages)</span></p>
<p>&nbsp;</p>

<p><span style="font-size:small">Create your own&nbsp;<strong><em>html5/responsive content under &ldquo;contentplaceholdermain&rdquo;</em></strong>&nbsp;and publish it.&nbsp;</span></p>
